## ChatGPT Powered Assistant Guide

### Which Model

**GPT-5.2 Instant**

> **Plan:** Free version of ChatGPT
> **Reason:** GPT-5.2 Instant is the default model available on the free plan. It is fast enough for day-to-day use and offers a reasonable balance between speed and output quality.
> **Note:** o1 reasoning models are intentionally avoided due to daily usage limits and slower iteration speed.
